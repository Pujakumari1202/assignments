## WebSockets
## Steps to run
 
# Initialise package.json

````bash
npm init -y
````

````bash
npm install typescript
````


## Initialise basic tsconfig.json
````bash
npx tsc --init
````

## then we do some changes in tsconfig.json file(Update tsconfig)
````bash
"rootDir": "./src",
"outDir": "./dist",
````


# create a src folder

## Run the ts file
````bash
npm run dev 
````


# Install ws
````bash
npm install ws @types/ws
````
